Mesh²

Mesh² is an open-source application for managing pcitures, videos and any other file, remotely without having to install any software.

The application is fully secured and users can manage permissions easily.

Features

Secured media server

  • OAuth 2.0 authentication through Google services
  • User IP filtering capability
  • Accessible other encrypted HTTPS

Directories

  • Navigate through directories
  • Create new directories
  • Rename directories
  • Download its content
  • Remove old ones

Images

  • View your prefered photos in a clear and efficient grid
  • Automatic pictures orientation based on camera meta data
  • Manual photos rotating by +90°, -90° or 180°
