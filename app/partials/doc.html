Mesh²

Mesh² is an open-source application for managing pcitures, videos and any other file, remotely without having to install any software.

The application is fully secured and users can manage permissions easily.

Features

Secured media server

  • OAuth 2.0 authentication through Google services
  • User IP filtering capability
  • Accessible over encrypted HTTPS

Directories

  • Navigate through directories
  • Create new directories
  • Rename directories
  • Download its content
  • Remove old ones

Images

  • View your prefered photos in a clear and efficient grid
  • Automatic pictures orientation based on camera meta data
  • Manual photos rotating by +90°, -90° or 180°

FAQ

What does Mesh² mean ?

Mesh² means Media Server For Humans.

Where are my files stored ?

Where you want in fact. Mesh² is a web-based user interface which helps you accessing your files where ever you stored them.

Any server with the backend API available will be compatible and accessible right here.

Can I manage my medias such as on my computer ?

You can create folders and organize your albums exactly the same way you would do it on your computer.

The difference is that your files are stored on a server where files can be automatically backed up and where your most beautiful pictures are accessible from any device connected to the Internet.
